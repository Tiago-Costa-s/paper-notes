# Paper Notes
O Paper Notes e um aplicativo web, que permite ao usuario criar notas de forma simples e rapida, ele possui um visual charmoso e bem intuitivo bem facil de mexer.

### Link do Projeto
link: https://paper-notes.vercel.app/

- CSS 3
- Javascript
